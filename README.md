# Page Analyzer

**Анализатор страниц** — веб-приложение для проверки сайтов на SEO-пригодность.  
Анализирует указанные URL, сохраняет результаты и показывает историю проверок.

## Демо

🔗 Приложение доступно по адресу: **[https://page-analyzer-bv9m.onrender.com](https://page-analyzer-bv9m.onrender.com)**

## Функциональность

- Добавление URL для анализа
- Проверка кода ответа сервера
- Извлечение SEO-тегов: `h1`, `title`, `description`
- История всех проверок
